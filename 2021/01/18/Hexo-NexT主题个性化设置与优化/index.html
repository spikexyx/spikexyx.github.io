Hexo-NexT主题个性化设置与优化

发表于 2021-01-18 分类于 建站日志 Valine：
本文字数： 549 阅读时长 ≈ 1 分钟

前言： 在成功建立网站之后，考虑到NexT主题的强大自定义功能和丰富的插件配置，进行个性化设置以使网站更加美观与个人化。本文主要附上几篇大佬的文章链接，大佬文章中已经介绍的非常详尽了，按需学习配置即可。

链接

  1. hexo的next主题个性化教程:打造炫酷网站——Moorez（非常非常详尽的教学，不过是2017年的教程，有少许地方现在已经不适用）

  2. Hexo-Next 主题安装并配置教程——苏寅 Blog（来自苏寅大佬的文章，而且实时更新，介绍了除了官方文档之外的一些插件与优化，简单明了且美观）

  3. 打造个性超赞博客 Hexo + NexT + GitHub Pages 的超深度优化——reuixiy（也是一篇深度好文）

  4. NexT官方文件网站

  5. Hexo官网

补充

  1. 最新版NexT主题中已经集成了Valine评论功能，只用在 主题 配置文件 _config.yml 中首先找到comment项将参数设为valine

    再找到 “valine” 项设置各项参数即可。注意其中的 appid 和 appkey 来自于leancloud上你开的app。

    （注意首先需要在leancloud上注册一个账号并新建一个app）（参考苏寅大佬文章中minivaline部分）

  2. 分享文章中改变主页背景图片的方法目前已失效，而且暂时没有找到合适的方法，考虑换背景图不是刚需，所以忽略。
